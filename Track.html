Big Data Engineer

AI Engineer / الذكاء الاصطناعي

Backend Python Developer

المهندس المسؤول عن بناء الأنظمة الخلفية للتطبيقات والمواقع، يتعامل مع قواعد البيانات، الخوادم، وواجهات الـ APIs لضمان الأداء، الأمان، والكفاءة باستخدام Python وأطرها مثل Django وFlask وFastAPI.

Backend JavaScript Developer (Node.js)

المهندس المسؤول عن بناء الأنظمة الخلفية للتطبيقات والمواقع باستخدام JavaScript أو TypeScript، يتعامل مع الخوادم، قواعد البيانات، والـ APIs اللي بترتبط بالـ Frontend لضمان الأداء، الأمان، والكفاءة. يستخدم Node.js وأطر مثل Express.js وNestJS لبناء APIs متقدمة وتطبيقات Real-Time وMicroservices.

LLM AI Developer (Large Language Models)

مهندس الـ LLM AI Developer هو الشخص اللي بيطوّر، يدرّب، ويستخدم النماذج اللغوية الضخمة (مثل GPT، Claude، LLaMA) لبناء أنظمة ذكية قادرة على فهم اللغة البشرية وتوليد محتوى ذكي في مجالات مختلفة (برمجة، تحليل، دعم فني، كتابة محتوى...). بيتعامل مع تدريب النماذج، Fine-tuning، Prompt Engineering، وبناء Chatbots وأنظمة توصية وتحليل نصوص. يستخدم أدوات زي LangChain، Hugging Face، OpenAI API، وبيتم دمج النماذج داخل التطبيقات والخدمات.

Cyber Security Engineer

مهندس الأمن السيبراني هو الشخص اللي بيحمي الأنظمة، الشبكات، وقواعد البيانات من أي اختراق أو تهديد إلكتروني. دوره الأساسي إنه يبني، يختبر، ويطوّر أنظمة حماية قوية تمنع القراصنة (Hackers) من الوصول غير المصرّح به، ويحافظ على سرّية وسلامة البيانات. بيشتغل على تحليل الثغرات، اختبار الاختراق (Penetration Testing)، تأمين الشبكات والسيرفرات، تشفير البيانات، ومراقبة الأنظمة لمنع أي نشاط مشبوه. بيستخدم أدوات زي Metasploit، Burp Suite، Nmap، Wireshark، Splunk، ويدمج حماية الأنظمة مع السحابة والتطبيقات.

Backend Java Developer

مهندس باك إند جافا هو الشخص اللي بيبني ويطوّر الجزء اللي ورا الكواليس في التطبيقات باستخدام لغة Java وبيئاتها زي Spring. دوره الأساسي إنه يصمم APIs، يدير قواعد البيانات، يطبق منطق الأعمال، ويضمن إن الأنظمة سريعة، آمنة، وقابلة للصيانة. بيشتغل على بناء خدمات مستقرة، تحسين الأداء، التعامل مع الرسائل (Kafka/RabbitMQ)، واستخدام السحابة (AWS, GCP, Azure) لنشر التطبيقات الكبيرة والمعقدة.

Machine Learning Engineer

مهندس تعلم الآلة هو الشخص اللي بيبني ويطوّر أنظمة قادرة على "التعلم من البيانات" بدل ما يتم برمجتها يدويًا. دوره الأساسي إنه يصمم النماذج (Models)، يدربها، ويحوّلها إلى أنظمة ذكية يمكن نشرها في الإنتاج — زي أنظمة التوصية، التنبؤ، والتصنيف. بيشتغل على تحليل ومعالجة البيانات (Data Preprocessing & Feature Engineering)، تدريب النماذج باستخدام مكتبات زي TensorFlow وPyTorch وScikit-learn، تقييم الأداء، وتحسين دقة النماذج. كما يقوم بنشر النماذج في التطبيقات والخدمات المختلفة ويطبق أفضل ممارسات MLOps لضمان الاستقرار والاعتمادية.
